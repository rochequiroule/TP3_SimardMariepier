Comment installer mon application TP3 - Projet final

  1. Installez l'application en cliquant sur le bouton ci-dessous. Après l'installation, le bouton est désactivé.

  2. Lorsque vous cliquez sur "Installer", une boîte de dialogue semblable à celles des plates-formes de téléchargement d'applications s'affiche.
  3. La boîte de dialogue inclut la "description" et les "captures d'écran" définies dans le fichier manifeste de l'application.
  4. Les captures d'écran doivent être différentes selon que l'application est installée sur un appareil mobile ou de bureau, en fonction de la valeur "form_factor" définie pour les captures d'écran dans le fichier manifeste
